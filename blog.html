4. Does it matter to use relative size(%,em,rem) one styling the size of an element so that the element is responsive ? Why ?

It is a big deal to use relative size because they help you to achieve a responsive and balanced design. While em is relative to the font-size of its direct or nearest parent, rem is only relative to the html (root) font-size. If a style sheet is built in a modular fashion, then rem shouldn’t be needed very often, but it can be handy at times. For example, if you’re trying to achieve consistent spacing without extra markup, rem can be used to define the padding and margins. When we look at em units the big problem here is that they are relative to the font size of their own element.Rem is helpful because of the ability to scale type across the entire page easily. On the other hand the main issue with using rem for font sizing is that the values are somewhat difficult to use which makes them inconvenient for making calculations.

6. Explain in detail(in technical terms) the specific changes in css3

Selectors

Using the new DOM selectors you can choose DOM elements based on their attributes. So you don't need to specify classes and IDs for every element. Instead, you can make use of the attribute field to style them.

Rounded Corners

Adjusting the height, width and positioning of rounded corner elements is a time consuming and laborious.Alteration in content can break them. CSS 3 addresses this problem by introducing the border-radius property, which gives you the same rounded-corner effect and you don't have to write all the code.

Border Image

Another feature in CSS 3 is the ability to replace a border with an image. The property border-image allows you to specify an image to display instead of a plain solid-colored border.

Box Shadows

This functionality enables a content to slide under another area such as footer.This process doesn't require javascript plugin

Transform (Element Rotation)

CSS 3 also introduced a property called transform, which enables rotating Web elements on a webpage. The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements. Many JavaScript extensions/plugins are available online for this feature, but they can make the code complicated and most importantly consume more resources.
